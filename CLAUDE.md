# klockslag.se

## Skrivregler (gäller all svensk text)

- Aldrig em-dash (—). Använd tankstreck `–` (en-dash) med mellanslag, eller skriv om utan streck.
- Följ Svenska skrivregler.
- Undvik AI-klichéer: "inte bara X utan också Y", tomma förstärkningsord, överdrivet parallella meningar, em-dash som övergång.
- Håll ton naturlig och konkret. Hellre kortare mening än pekpinnar.

Gäller allt: copy, JSON-data, kommentarer, commit messages, svar i chatten.
